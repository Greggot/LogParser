# LogParser TRW & DAF

Приложеня фильтруют данные из логов формата .txt. Запуск:
`ParserTRW <./PathToLog/Input.txt> <./PathToBin/Output.bin>` или `ParserDAF <./PathToLog/Input.txt> <./PathToBin/Output.bin>`.
